Enumberable#cycle

`Enumberable#cycle` is a powerful method that can be use in a variety of ways. At first glance it operates a lot like `each` but it actually extends beyond that. `cycle` iterates through an array or a hash and passes each element (or key/value pair) to a block of code. If a limit or `break` condition is not supplied, it will cycle for ever.
